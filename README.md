# 📊 Bank Customer Churn Analysis

## 📌 Overview
This project focuses on analyzing customer churn in a banking dataset and identifying key factors that influence customer retention.

---

## 📂 Dataset
The dataset contains customer information such as:
- Credit Score
- Geography
- Gender
- Age
- Balance
- Active Status
- Churn (Exited)

---

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)

---

## 🚀 Day 1 Progress - Data Understanding
- Loaded dataset using Pandas
- Checked data structure and types
- Verified missing values
- Analyzed target variable (Exited)
- Created initial visualization

---

## 📊 Key Insights (Day 1)
- Dataset contains ~10,000 records
- No missing values found
- Dataset is imbalanced (more non-churn customers)
